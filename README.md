# IoT WEATHER STATION

## PROJECT DESCRIPTION
This project is an IoT-based weather monitoring system developed using ESP32. 
It uses a DHT22 sensor to measure temperature and humidity and displays the readings on an OLED display.

## COMPONENTS USED
- ESP32 Microcontroller
- DHT22 Temperature and Humidity Sensor
- OLED Display (128x64)
- Breadboard
- Jumper Wires

## SOFTWARE USED
- Arduino IDE
- Wokwi Simulation

## WORKING PRINCIPLE
The DHT22 sensor collects environmental temperature and humidity data.
The ESP32 processes the sensor values and displays the real-time readings on the OLED display.
The values can also be monitored through the Serial Monitor.

## FEATURES
- Real-time temperature measurement
- Humidity monitoring
- OLED display output
- Sensor-based weather monitoring
- Embedded system implementation
